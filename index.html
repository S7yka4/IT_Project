Для вас представлен "Продукт для сборки ПК"

Для чего нужен наш продукт?
В наше время компьютер выполняет колоссальный спектр задач, а на рынке существует огромное количество готовых вариантов, но что делать если ни один из компьютеров, работающих из коробки, не подходит конкретно вам? Представляем вам наш продукт, позволяющий собрать компьютер из комплектующих, который подойдет именно вам.

Цель проекта сделать достаточно сложный процесс подбора комплектующих для ПК проще и удобнее, а также проект включает в себя базу данных компьютерных комплектующих, благодаря которой у вас всегда будет возможность ознакомиться с актуальными характеристиками комплектующих ПК в структурированном виде.

Продукт представляет из себя комплекс программ, включающий СУБД, публичное API и desktop клиент. Достаточно просто скачать клиент и подключиться к интернету. Клиент свяжется с базой данных и предложит комплектующие для сборки. При выборе одного компонента будет изменяться выдача компонентов других типов, чтобы обеспечить подбор совместимых компонентов.

За основу взята концепция из мультика "Бивис и Баттхед" 1993 года чтобы упростить сборку ПК и сделать её развлекательным занятием.

Разработанные Технологии от Бивиса и Баттхеда

WEB-API приложение

API позволяет связывать различные системы друг с другом – например, аккаунты различных сервисов, банковские аккаунты и платежные системы. Обеспечение безопасности данных. Ряд функций выносится в отдельное приложение, что делает невозможным их некорректное использование

Свой Клиент

Аппаратный или программный компонент вычислительной системы, посылающий запросы серверу Клиент — это программа, обеспечивающая взаимодействие с сервером. Клиент отправляет запросы серверу, и получает результат обработки этих запросов от сервера. Так или иначе клиент служит для доступа к ресурсам сервера.

Своя База Данных

База данных — это организованная структура, которая предназначена для хранения и обработки взаимосвязанной информации. Ее использование позволяет уменьшить нагрузку на сервер, увеличивается скорость работы с данными и контентом страниц, а так же увеличивается безопасность сайта, что немало важно.
